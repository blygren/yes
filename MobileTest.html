Game & Hints

Tool Hints
  • Spawn Tool: Tap screen to create cubes.
  • Drag Tool: Touch and drag cubes to move them.
  • Slingshot Tool: Touch a cube, pull back, and release to launch.
  • Tether Tool: Tap one cube, then another to connect. Tap a tether line to remove it.
  • Explosion Tool: Tap screen to create an explosion.
  • Random Tool: Tap screen to apply random forces/colors.
  • Multi-Cube Spawner: Tap screen to spawn several cubes.
  • Blade Tool: Tap a cube to split it.
General Game Info
  • Double-tap the screen to pause/resume.
  • Use the menus to customize physics, cube properties, and visuals.
  • Try enabling 'Use Device Tilt for Gravity' in Physics settings!
  • Explore different tool modes and settings to unlock achievements.
